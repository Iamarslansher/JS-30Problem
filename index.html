Unique JavaScript tidbits

  • JavaScript === not Java: Same name, different languages ‚Äî JS was created in 10 days (1995) by Brendan Eich.
  • First-class functions: Functions can be stored in variables, passed around, and returned like values.
  • Prototypal inheritance: Objects inherit directly from other objects (no classical classes needed).
  • Event loop magic: Callbacks, microtasks and macrotasks decide execution order ‚Äî async is subtle!
  • Falsy values: 0, '', null, undefined, NaN, and false are all falsy ‚Äî helpful in conditions.
  • Coercion quirks: '5' - 2 -> 3 (string to number), but '5' + 2 -> '52' (concatenation).
  • Useful tools: Console.table(), debugger, and structuredClone() (modern deep-clone).
